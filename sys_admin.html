Homework

  1. List different attributes, such as package manager, price, and target audience for different Linux distributions
  2. Gain practice with commands like cd, ls, find, grep, du, wc, head, tail, and tar as well as gain a general understanding of the format of bash commands
  3. Gain experience in account management by creating accounts on a Fedora VirtualBox and editing priviledges, setting up SSH, and modifying skeleton directories
  4. Write 2 bash scripts:
    1. Backup all files within a user's home directory that have been edited within the last 24 hours and create an optional argument to install a cron job to automate this task. Make sure to compress the files into a tarball in a hidden directory
    2. Get the information on a single file such as format, size, and compression ratio based on user specified arguments
  5. Add all commands learned to a cheat sheet throughout the semester.

Projects

  1. Install a JupyterHub server on a Fedora VirtualBox instance with the following configurations:
    • Make the server a systemd service so it can start up on boot of the instance
    • Make sure Google OAuth is working correctly to only allow users with our school email domain
    • Create a shared directory for all JupyterHub users but only allow each user access to their own files (sticky bit)
    • Install more packages onto the server and have it run on https instead of http
    • Install more languages onto the server instead of only Python3
  2. Set up a network of computers and printers using AD DC, DNS, NFS, DHCP, Kerberos, and Samba
    • 1 Fedora server, 1 Fedora Workstation, and 1 Windows machine
    • Make sure users can access there files on all computers and print from all computers
